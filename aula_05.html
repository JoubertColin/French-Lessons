3. 

Associez les éléments du premier tableau à ceux du second pour compléter correctement les phrases :

Première Tableau

  • Pour être ponctuel / ponctuelle, il faut
  • Pour être rigoureux / rigoureuse et organisé(e), il faut
  • Pour être rapide ou efficace, il faut
  • Pour être polyvalent(e), il faut
  • Pour avoir l'esprit d'équipe, il faut
  • Pour avoir une bonne relation avec les clients, il faut

Deuxième Tableau

  • remplir les rayons vite, car le magasin doit être prêt à l'ouverture.
  • répondre aux clients, être poli, serviable.
  • collaborer avec les collègues.
  • commencer tôt le matin, respecter les horaires.
  • savoir faire différentes tâches (rangement, accueil, nettoyage).
  • vérifier les dates de péremption, ranger les produits correctement.
Cliquez sur un élément de chaque liste pour former une phrase.
